Вклад к курс

Вы можете помочь сделать курс лучше и внести в него вклад! Есть два пути:

  1. Создать Issue с вашими предложениями, вопросами или комментариями для их обсуждения
  2. Создать Pull Request с внесением изменений

Предложение изменений с Pull Request

Для больших изменений лучше начать с обсуждения в Issue, а небольшие изменения, например, исправление опечатки, можно сразу предложить в Pull Request.

Начать лучше с прочтения раздела с технической информации о разработке курса на странице > DEVELOPMENT.

  1. Создайте свой Fork репозитория
  2. Склонируйте свой репозиторий
  3. Внесите изменения
  4. Если требуется, соберите HTML и PDF версии изменённых страниц или материалов
  5. Сделайте commit; в сообщении кратко опишите предлагаемое изменение (см. примеры коммитов в истории)
  6. Отправьте изменения с push в свой fork
  7. Создайте Pull Request и опишите ваши изменения
  8. Ждите ^_^
